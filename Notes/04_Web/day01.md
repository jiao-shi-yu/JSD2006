### HTML
Hyper Text Markup Language，超文本标记语言。
- 超文本：不仅仅是纯文本，还包括，字体效果和多媒体信息。
- 标记语言：由一对尖括号组成
- 和 XML 的区别：XML是可扩展标记语言。标签可以任意写，没有限制。
- 学习 HTML 就是学习有哪些标签，这些标签怎么用。
