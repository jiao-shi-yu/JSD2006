
先学一个 Scanner 扫描器类。

## Scanner接收用户输入的数据。
1. 导包
```java
import java.util.Scanner;
```

2. 实例化一个 scanner 对象。
```java
Scanner scanner = new Scanner(System.in);
```

3. 扫描用户输入
```java
System.out.println("请输入年龄:");
int age = scanner.nextInt();
System.out.println("请输入价格");
double price = scanner.nextDouble()
```
