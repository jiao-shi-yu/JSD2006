### 能不能用循环嵌套的方式？
例如：
```java
public void start() {
    try {
        while (true) {
            System.out.println("等待客户端连接");
            Socket socket = serverSocket.accept();
            System.out.println("一个客户端连接成功");
            
            InputStream in = socket.getInputStream();
            InputStreamReader isr = new InputStreamReader(in, "UTF-8");
            BufferedReader br = new BufferedReader(isr);
            
            String message = null;
            
            while((message = br.readLine()) != null) {
                
                System.out.println("客户端说：" + message);
            }
        }
    } catch (Exception e) {
        // TODO: handle exception
    }
}
```

不行。因为内层是一个死循环。一直接受第一个客户端的消息，不会跳出内层循环。除非第一个客户端断开连接。br 读到 null.

## 服务端向多个客户端发送消息

用一个数组，存储各个 ClientHandler 的输出流。
通过它来转发消息。
```java
private PrintWriter[] allOut = {};
```


在 ClientHandler 的 run() 方法里面，给客户端发送信息。

## 客户端接收服务端的信息

客户端向服务器发送消息 和 客户端接收服务器的消息。应该是两个独立的事件。可以用两个线程来分别执行。

```java
    private class ServerHandler implements Runnable{
        public void run() {
            // 获取输入流，以读取服务端发送过来的消息。
            try {
                InputStream in = socket.getInputStream();
                InputStreamReader isr = new InputStreamReader(in, "UTF-8");
                BufferedReader br = new BufferedReader(isr);
                
                // 一直读取服务器的消息
                String line = null;
                while ((line = br.readLine())!=null) {
                    System.out.println(line);
                }
                
            } catch (IOException e) {
                // TODO Auto-generated catch block
                e.printStackTrace();
            }
        }
    }
```

## ClientHandler中添加一个属性。用以区分客户端。
```java
class ClientHandler implements Runnable {
    private Socket socket;
    private String host;
    public ClientHandler(Socket socket) {
        this.socket = socket;
        host = socket.getInetAddress().getHostAddress();
    }
    public void run() {
        try {
            
            // 获取输出流
            OutputStream out = socket.getOutputStream();
            PrintWriter pw = new PrintWriter(out, true); // <---
            
            // 获取输入流
            InputStream in = socket.getInputStream();
            InputStreamReader isr = new InputStreamReader(in, "UTF-8");
            BufferedReader br = new BufferedReader(isr);
            
            
            // 将 printWriter 添加到数组中
            // 1. 扩容
            allOut = Arrays.copyOf(allOut, allOut.length+1);
            // 2. 赋值
            allOut[allOut.length-1] = pw;
            
            
            // 读取对应客户端的消息，向每一个客户端转发消息。
            String message = null;
            while((message = br.readLine()) != null) {
                System.out.println(host + ": "  + message);
                for (PrintWriter writer: allOut) {
                    writer.println(host + ": "  + message);
                }
            }
        } catch (Exception e) {
            e.printStackTrace();
        } 
    }
}
```
## 移除无用的PrintWriter，并显示当前在线人数。
```java
    class ClientHandler implements Runnable {
        private Socket socket;
        private String host;
        public ClientHandler(Socket socket) {
            this.socket = socket;
            host = socket.getInetAddress().getHostAddress();
        }
        public void run() {
            PrintWriter pw = null;
            try {
                
                // 获取输出流
                OutputStream out = socket.getOutputStream();
                pw = new PrintWriter(out, true); // <---
                
                // 获取输入流
                InputStream in = socket.getInputStream();
                InputStreamReader isr = new InputStreamReader(in, "UTF-8");
                BufferedReader br = new BufferedReader(isr);
                
                
                // 将 printWriter 添加到数组中
                // 1. 扩容
                allOut = Arrays.copyOf(allOut, allOut.length+1);
                // 2. 赋值
                allOut[allOut.length-1] = pw;
                
                // 显示连接到服务器的新客户端，显示当前在线人数
                System.out.println(host + " 上线了，当前在线人数：" + allOut.length);
                
                
                
                
                // 读取对应客户端的消息，向每一个客户端转发消息。
                String message = null;
                while((message = br.readLine()) != null) {
                    System.out.println(host + ": "  + message);
                    for (PrintWriter writer: allOut) {
                        writer.println(host + ": "  + message);
                    }
                }
            } catch (Exception e) {
                e.printStackTrace();
            } finally {
                remove(pw);
                System.out.println(host + " 下线了，当前在线人数：" + allOut.length);
                try {
                    socket.close();
                } catch (IOException e) {
                    e.printStackTrace();
                }
            }
        }
        
        private void remove(PrintWriter printWriter) {
            for (int i = 0; i < allOut.length; i++) {
                if (printWriter.equals(allOut[i])) {
                    allOut[allOut.length-1] = allOut[i];
                }
            }
            allOut = Arrays.copyOf(allOut, allOut.length-1);
        }
    }
```
## 使用 synchronized 关键字，确保线程安全
多个线程都会对 allOut 进行操作，因此 allOut 是临界资源。
当某个线程删除 allOut 中某个元素时，其他线程不应该遍历。
所以要对操作 allOut 的使用同步块，锁对象选择临界资源 allOut 就行。

```java

public void run() {
    PrintWriter pw = null;
    try {
        
        // 获取输出流
        OutputStream out = socket.getOutputStream();
        pw = new PrintWriter(out, true); // <---
        
        // 获取输入流
        InputStream in = socket.getInputStream();
        InputStreamReader isr = new InputStreamReader(in, "UTF-8");
        BufferedReader br = new BufferedReader(isr);
        
        synchronized(allOut) {
            // 将 printWriter 添加到数组中
            // 1. 扩容
            allOut = Arrays.copyOf(allOut, allOut.length+1);
            // 2. 赋值
            allOut[allOut.length-1] = pw;
            
        }
    
        // 显示连接到服务器的新客户端，显示当前在线人数
        System.out.println(host + " 上线了，当前在线人数：" + allOut.length);
        
        
        
        
        // 读取对应客户端的消息，向每一个客户端转发消息。
        String message = null;
        while((message = br.readLine()) != null) {
            System.out.println(host + ": "  + message);
            synchronized(allOut) {
                for (PrintWriter writer: allOut) {
                    writer.println(host + ": "  + message);
                }
            }
        }
    } catch (Exception e) {
        e.printStackTrace();
    } finally {
        synchronized(allOut) {                  
            remove(pw);
        }
        System.out.println(host + " 下线了，当前在线人数：" + allOut.length);
        try {
            socket.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
```
